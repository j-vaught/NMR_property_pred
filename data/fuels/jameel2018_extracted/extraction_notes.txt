FUNCTIONAL GROUP WEIGHT PERCENT FORMULAS (Table 2)
==================================================
Given in Table 2 of paper. For each group, three steps:
  1. Compute mol% from integral intensity ratio
  2. Compute arbitrary weight using MW of group and number of H atoms
  3. Compute weight% from all groups

Example for paraffinic CH2:
  M_PCH2 = (H + D)/T * 100       (mol %)
  G_PCH2 = M_PCH2 * 14/2         (arb. weight units)
  W_PCH2 = G_PCH2 / (sum of all G values) * 100   (weight %)

BRANCHING INDEX EQUATIONS
=========================
For pure compounds (Eq 1):
  BI = (no. of C atoms connected to longest chain + sum of PI for each) / no. of C atoms in longest chain

Position Index (PI):
  PI = 0        if C at outermost position
  PI = 0.5      if C 1 position from outermost
  PI = 0.5+0.5  if C 2 positions from outermost, etc.

For gasolines from NMR (Eq 5):
  BI_gasoline = (I/3 + J/3 + K/2 + H/2) / (4*A/T + 0.5*10*F/T + E/3 + (L+M)/3 + D/2 + C)

MLR EQUATIONS (for comparison; ANN is the primary model)
========================================================
RON = 44.82 + 0.86*CH3 + 0.25*CH2 + 0.23*CH + 0.76*olefinic - 0.43*naphthenic
      + 0.56*aromatic + 1.22*ethanolic_OH - 0.31*MW + 26.69*BI          (Eq 6)

MON = 31.26 + 0.86*CH3 + 0.29*CH2 + 0.26*CH + 0.49*olefinic + 0.40*naphthenic
      + 0.49*aromatic + 0.94*ethanolic_OH - 0.22*MW + 28.47*BI          (Eq 7)

Note: MLR R2 = 0.52 (RON) and 0.51 (MON) - poor fit; ANN is much better (R2 = 0.99)

ANN MODEL ARCHITECTURE (Table 6)
=================================
RON model: 9-540-314-1  (9 inputs, hidden layer 1: 540 nodes, hidden layer 2: 314 nodes, 1 output)
  - Mean absolute error: 1.6, RMS error: 2.2, Mean % error: 1.8

MON model: 9-340-603-1  (9 inputs, hidden layer 1: 340 nodes, hidden layer 2: 603 nodes, 1 output)
  - Mean absolute error: 1.3, RMS error: 2.2, Mean % error: 1.6

9 Input parameters: paraffinic CH3 (wt%), paraffinic CH2 (wt%), paraffinic CH (wt%),
  olefinic CH=CH2 (wt%), naphthenic CH-CH2 (wt%), aromatic C-CH (wt%),
  ethanolic OH (wt%), molecular weight (g/mol), branching index (BI)

Training/test split: 224 training / 57 test (80/20 split)
Octane sensitivity mean absolute error: 1.4

FACE GASOLINE IDENTIFIERS
=========================
Six FACE gasolines used: A, C, F, G, I, J
(FACE = Fuels for Advanced Combustion Engines)

WHAT IS NOT IN THE PAPER
=========================
- The actual NMR functional group weight fractions for each compound/blend are NOT tabulated.
  For pure compounds, these must be computed from molecular structure.
  For blends, they are computed from composition and pure component properties.
  For FACE gasolines, they were measured via NMR but the spectra/values are not provided.
- ANN model weights are not provided (the paper references CloudFlame web tool).
- The Supplementary Information only describes ANN methodology, not additional data.

=============================================
